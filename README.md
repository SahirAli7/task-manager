# Task Manager

A web-based task manager application that lets users add, complete, and delete tasks with a clean, responsive interface.

## Features

- Add new tasks
- View task list
- Mark tasks as completed / undo completion
- Delete tasks
- Empty state message when no tasks exist
- Responsive design for mobile and desktop

## Technologies Used

- HTML5
- CSS3 (Flexbox, Gradient Backgrounds, Transitions)
- JavaScript (DOM Manipulation, Event Handling)

## Steps to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/SahirAli7/task-manager.git
   ```
2. Navigate to the project folder:
   ```bash
   cd task-manager
   ```
3. Open `index.html` in your browser.

Or visit the live hosted version at: [Live URL](https://SahirAli7.github.io/task-manager/)
